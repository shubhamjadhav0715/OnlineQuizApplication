## 🚀 How to Run on Localhost

### **Quick Method (5 Minutes):**

1. **Clone Repository**
   ```bash
   git clone https://github.com/shubhamjadhav0715/OnlineQuizApplication.git
   ```

2. **Setup Database**
   - Execute `Database/QuizDB.sql` in SQL Server

3. **Open in Visual Studio 2022**
   - Right-click `Default.aspx` → Open with Visual Studio 2022

4. **Set Start Page**
   - Right-click `Default.aspx` → Set As Start Page

5. **Run**
   - Press **F5**

### **Detailed Guide:**
📖 See [LOCALHOST_SETUP.md](LOCALHOST_SETUP.md) for complete step-by-step instructions

## 🔧 Customization Guide

### **Change Colors:**

Edit `site.css` root variables:
```css
:root {
    --primary-gradient: linear-gradient(135deg, #YOUR_COLOR_1, #YOUR_COLOR_2);
    --primary-color: #YOUR_PRIMARY_COLOR;
}
```

### **Modify Animations:**

Adjust animation duration:
```css
:root {
    --transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
}
```

### **Change Fonts:**

Update font family:
```css
body {
    font-family: 'Your Font', 'Segoe UI', sans-serif;
}
```

## 🐛 Known Issues & Solutions

### **Issue: CSS not loading**
**Solution:** Clear browser cache (Ctrl + Shift + Delete) and hard refresh (Ctrl + F5)

### **Issue: Database connection failed**
**Solution:** Update connection string in Web.config to match your SQL Server instance

### **Issue: Build errors**
**Solution:** Right-click App_Code → Properties → Build Action → Compile

📖 **Full troubleshooting guide:** See [LOCALHOST_SETUP.md](LOCALHOST_SETUP.md)
