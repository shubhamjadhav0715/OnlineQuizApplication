Hi! Welcome to your Online Quiz Application.

FOLLOW THESE SIMPLE STEPS TO RUN THE APPLICATION:

================================================================================
STEP 1: CREATE DATABASE
================================================================================

1. Open SQL Server Management Studio (SSMS)
   OR
   Open Visual Studio 2022 → View → SQL Server Object Explorer

2. Connect to: (localdb)\MSSQLLocalDB

3. Execute the SQL file:
   - File → Open → File
   - Select: Database\QuizDB.sql
   - Click Execute (F5)

4. Verify: You should see "Database created successfully!"

================================================================================
STEP 2: OPEN IN VISUAL STUDIO 2022
================================================================================

EASIEST WAY:
1. Open File Explorer
2. Go to: OnlineQuizApplication\OnlineQuizApplication\
3. Find: Default.aspx
4. Right-click on Default.aspx
5. Click: "Open with Visual Studio 2022"

================================================================================
STEP 3: SET START PAGE
================================================================================

1. In Solution Explorer (right panel), find Default.aspx
2. Right-click on Default.aspx
3. Click: "Set As Start Page"

================================================================================
STEP 4: RUN THE APPLICATION
================================================================================

Press F5 on your keyboard!

Your browser will open with the application.

COMMON ERRORS & FIXES
================================================================================

ERROR: "Please select a valid startup item"
FIX: Right-click Default.aspx → Set As Start Page → Press F5

ERROR: "Database does not exist"
FIX: Re-run Database\QuizDB.sql

ERROR: "Build failed"
FIX: Right-click App_Code folder → Properties → Build Action → Compile
